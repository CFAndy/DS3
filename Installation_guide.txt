This document is a guide for installing required software.
Please follow these steps to install Anaconda and related packages.

1) Download and Install Anaconda from https://www.anaconda.com/download/ (Use Python 3.6 version)
2) After installation, run the Anaconda prompt as administrator.
3) Then, used the following command:
	conda install <Package names>

Package names:
 - simpy
 - pandas
 - matplotlib
 - networkx
